# What is Visual Basic?

    Visual Baisc 是 Microsoft  公司推出的可视化开发环境，是 Windows 下最优秀的程序开发工具之一。利用 Visual Basic 可以开发出具有良好交互功能、良好的兼容性和拓展性的应用程序
